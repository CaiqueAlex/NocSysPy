# Compilador NocSys

Compilador educacional para a linguagem NocSys (PySimple), desenvolvido para a disciplina de Compiladores e Linguagens Formais.

## Linguagem NocSys
NocSys é uma linguagem de programação educacional inspirada em Python, com sintaxe simplificada.

## Estrutura do Projeto
- `/docs` - Documentação e especificações
- `/src` - Código fonte do compilador
- `/examples` - Exemplos de programas em NocSys

## Como usar
```python
python src\compilador.py examples\hello_world.pys
