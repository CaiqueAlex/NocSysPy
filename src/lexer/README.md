# Analisador Léxico - NocSysPy

Este módulo contém a implementação do analisador léxico baseado em Autômatos Finitos Determinísticos (AFD).

## Estrutura
- `dfa.py`: Implementação da máquina de estados e definições dos autômatos.
- `lexer.py`: Motor principal que processa o código fonte usando os AFDs.

## Como Executar

Na raiz do projeto, execute o script principal:

```bash
python3 main.py
